# spooky_go 🎃👻

Rust and Python library for the game of Go.

# Features

- Supports variable board sizes.
- Relatively fast.
- Out-of-the-box support for DL/ML (action encoding and decoding methods).

# Install

## Rust

```fish
cargo add spooky_go
```

## Python

```fish
uv add spooky-go
```

Includes type hints.

# Develop

### Tests

- `fish run_tests.fish`
    - `fish run_python_tests.fish`
    - `fish run_rust_tests.fish`

### Lints

- `fish run_lints.fish`

### Performance

- `fish run_benchmark.fish`
